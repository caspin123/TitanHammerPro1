# ⚒️ TitanHammer Pro

Professional Minecraft Plugin for Paper 1.21+

## Features
✅ Custom Netherite Axe (CustomModelData 2025)
✅ Level & XP System
✅ Range Mining (3x3 at Level 5+)
✅ Particle & Sound Effects
✅ Statistics Tracking

## Installation
1. Download JAR from GitHub Actions → Artifacts
2. Place in plugins/ folder
3. Restart server

## Commands
- `/th give` - Get Titan Hammer
- `/th stats` - View your stats
- **Shift + Right Click** - Quick stats

## Permissions
- `titanhammer.use` - Use the plugin (default: true)
- `titanhammer.give` - Give hammers (default: op)
